## 简介

`mcsmapi` 是一个基于 [MCSManager](https://github.com/MCSManager/MCSManager) 的 Pypi 包，旨在简化与 MCSM API 的交互。

通过这个库，您可以更轻松地访问和操作 MCSM 提供的各种功能。

## 安装

```bash
pip install mcsmapi
```

如果你需要最新的构建文件，请访问
[Actions](https://github.com/molanp/mcsmapi/actions)

## 支持的 MCSM 版本

| MCSM 版本 | 支持状态 |
| :-------: | :------: |
|   10.x    |    ✅    |
