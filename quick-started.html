MCSM API ​

简介 ​

mcsmapi 是一个基于 MCSManager 的 Pypi 包，旨在简化与 MCSM API 的交互。

通过这个库，您可以更轻松地访问和操作 MCSM 提供的各种功能。

安装 ​

bash
pip install mcsmapi
1

如果你需要最新的构建文件，请访问 Actions
